# Pokepedia 🐾

Pokepedia is your ultimate guide to the Pokémon universe! Built using **Flutter** and **Firebase**, this application serves as a complete directory for Pokémon fans. It contains the real names, stats, and detailed information about every actual Pokémon. 

Whether you are a casual fan or a Pokémon master, Pokepedia has everything you need to know about your favorite Pokémon.

### 🌟 Key Features
* **Comprehensive Pokédex:** Explore the real names and detailed descriptions of every Pokémon.
* **Secure Authentication:** User-friendly Login and Sign-Up screens powered by Firebase Authentication.
* **Beautiful Animations:** Engaging UI with smooth Lottie animations.
* **Responsive Design:** Optimized for a seamless experience on different screen sizes without overflow issues.

### 🛠️ Tech Stack
* **Frontend:** Flutter, Dart
* **Backend:** Firebase Authentication & Cloud Firestore
* **Other Tools:** Lottie for animations.
